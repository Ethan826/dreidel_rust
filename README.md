# Dreidel

Dreidel is fun, but it takes too damned long to play. Now, all the thrills, but
without the time commitment.

## Setting up

### Install Rust

```bash
curl https://sh.rustup.rs -sSf | sh
```

### Clone this repo

```bash
git clone https://github.com/Ethan826/dreidel_rust
```

### Build and run

```bash
cd dreidel_rust
cargo run --release
```
